Linear Gradient

Gradient Position is optional, default is top. Position can be a degree (90deg). Mixin supports up to 10 color-stops.

{% highlight scss %} @include linear-gradient(#1e5799, #2989d8); @include linear-gradient(top, #8fdce5, #3dc3d1); @include linear-gradient(50deg, #1e5799 0%, #2989d8 50%, #207cca 51%, #7db9e8 100%); {% endhighlight %}
