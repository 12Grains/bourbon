Transitions

Shorthand mixin: Supports multiple parentheses-delimited values for each variable.

Demo Code

{% highlight scss %} @include transition (all, 2.0s, ease-in-out); @include transition ((opacity, width), (1.0s, 2.0s), ease-in, (0, 2s)); @include transition ($property:(opacity, width), $delay: (1.5s, 2.5s)); {% endhighlight %}
