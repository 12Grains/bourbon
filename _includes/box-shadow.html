Box Shadow

Box-shadow supports single or multiple arguments.

Single Box Shadow

{% highlight css %} @include box-shadow(0 0 3px 0 #ff0000); {% endhighlight %}

Multiple comma-delimited Box Shadows

{% highlight css %} @include box-shadow(1px 1px 2px 0 #ff0000, -1px -1px 2px 0 #ccc); {% endhighlight %}
