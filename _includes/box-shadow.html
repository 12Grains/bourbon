Box Shadow

Box-shadow supports single or multiple arguments.

Single Box Shadow

{% highlight scss %} div { @include box-shadow(0 0 10px 0 black); } {% endhighlight %}

Multiple comma-delimited Box Shadows

{% highlight scss %} @include box-shadow(1px 1px 10px 0 green, -1px -1px 10px 0 blue); {% endhighlight %}
