Flex Box

The flex-box mixin is bases on the 2009 w3c spec. The mixin with change to the flexible box-model. more info

Demo Code

{% highlight scss %} div.parent { @include display-box; @include box-align(start); @include box-orient(horizontal); @include box-pack(start); } div.parent > div.child { @include box-flex(2); } # Alternative custom shorthand mixin. div.parent { @include box($orient: horizontal, $pack: center, $align: stretch); } {% endhighlight %}
